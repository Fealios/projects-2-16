## Scrabble Game

#### Specs

* Take a word as input and return word
  * **Input:** "hello"
  * **Output:** "hello"


* Take word and return letters
  * **Input:** "hello"
  * **Output:** "h" "e" "l" "l" "o"


* Take word and return values
  * **Input:** "hello"
  * **Output:** "4" "1" "1" "1" "1"


* Take word and return score
  * **Input:** "hello"
  * **Output:** 8
